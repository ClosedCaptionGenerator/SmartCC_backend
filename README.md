# SmartCC_backend
> This is the backend of the SmartCC project.

## Installation

```
git clone
set .env ('/' and '/webapp')
```

## How to start

### active venv
```
python3 -m venv venv
source venv/bin/activate
```

### Install requirements
```
pip install -r requirements.txt
```

### migrate & runserver
```
python app.py
```
